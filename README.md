### DESCRIPTION
A website I made for calculating Elo Ratings between me and my friends.
Quickly see current rankings, win streaks and amount of matches for each player.
Also useful for keeping track of historical matches and Head-to-Head standings.
Data is stored in localstorage and can be exported and imported easily.

### FEATURES
#### Players
- View current rankings and amount of matches played
- See which players have winstreaks (no effect on ratings)
- Also an overview of all players where you can:
    - See all ranked and unranked players sorted by games played 
    - Quickly edit names
    - Hide inactive players (but keep all their data)
#### Data Entry
- Add new players by entering their name and choosing starting elo:
    - Beginner: 600
    - Intermediate: 800
    - Advanced: 1000
- Add new matches easily by selecting:
    - Player 1
    - Player 2
    - Winner (or draw)
- Export and import data
- Reset all data (export first to have a backup)
#### Match History
- Chronological view of all historical matches with:
    - Players names
    - Gained/Lost elo
    - Exact elo changes
- Can be filtered by player
#### Head-to-Head standings
- Head-to-Head scores for every player combo
- Can be filtered by player
